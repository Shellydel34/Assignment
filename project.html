What is Git

Git is a version control system.It is good for tracking code changes and manages projects with repositories.

Steps to take to have a functional Git

Install Git:
For windows download gitbash from your web browser and install.
Configure Git:
Let git know who you are by using git config --global user.name "your user name"to set your user name global for every repo and also use git config --global user.email "your email"to set your email global for every repo.
Create a github account:
Search github.comgithub.com on your browser and create an account by inputing the required things and verify your email to complete the process.After that,name your repo,input a descriptionabout your repo,click on the README option and under the licence option click on MIT licence and create.
Clone your repo with gitbash:
copy the urlon your repo and paste it on your gitbash after typing the clone command "git clone URL"
Clone your repo with Gitbash:
Create a folder on VScode using html and name it after your repo.Under that folder right click to craete a file and name it index.html
Initialize Git
Set git on the folder by using the init command.

Git have commands that manages projects with repositories and track code changes. An image of some basic git commands will be shown below.

Explanations

Git Commit:
(git commit -m "A message")commits keep track of our progress and changes.When we commit, we should always put a message because it helps us to see what has been changed and when.
Git Add:
(git add .)It adds all modified and new files in the current directory.
Git Push
(git push) It pushes local commit to a remote repository.
git clone
(git clone url) It creates a local copy of a remote repository.
Git Pull
(git pull -u origin) Fetches changes from a remote repository and merges them into the current barnch.
Git Log
(git commit log) It displays commit history or history of other git commands.
Git Remote
(git remote -v) It lists remote repositories associated with your local repo.
Git Status
(git status) It shows the status of your working directory and staged changes.
Git Diff
(git diff)It shows diff of what is changed but not staged.
Git Init
(git init)It initiates git to a folder or file.

What is Vscode

Visual studio code is a code editor redefined for building and debugging modern web and cloud applications.

Steps to take to have a functional vsc

Install Vsc
use your prefered browser, search for vsc and download.
Install Extensions
Extensions let you add languages,debuggers,and tools to your installation to support your development workflow.click on the settings option on the left hand corner of your vscode,you will see an extension option. Tap on it and download the necessary extensions needed.
